Privacy Policy

Traxis Label Printer — Chrome Extension

Effective date: April 30, 2026

Publisher: Traxis Manufacturing LLC, Austin, TX

What This Extension Does

Traxis Label Printer adds a print-label button to pages within our company's ProShop ERP system. When clicked, it reads product data from the current page and sends a label image to a Brother label printer on our local network. That is the extension's only function.

Data We Collect

None. This extension does not collect, store, transmit, or share any personal data or browsing information. Specifically:

  • No analytics, telemetry, or tracking of any kind.
  • No cookies are set by the extension.
  • No data is sent to any third-party server, cloud service, or external endpoint.
  • No user accounts, registration, or sign-in.

Data the Extension Reads

The extension reads product information (work order numbers, material descriptions, part numbers, tool IDs, equipment names) solely from pages on our company's private ProShop ERP instance. This data is used only to render a label image in the browser tab and is never stored or transmitted beyond the local network print service.

Network Communication

The extension communicates with exactly two hosts, both declared in the manifest:

  • traxismfg.adionsystems.com — Our company's ProShop ERP instance (reads page data and, for some label types, queries the ProShop GraphQL API using the user's existing session cookie).
  • 10.1.1.242:5002 — A Brother label printer service on our private local network. Label images are sent here for printing. This address is not reachable from the public internet.

Permissions

  • activeTab — Required to read product data from the current ProShop page when the user clicks the print button.
  • Host permissions — Scoped to our ProShop domain and local printer. The extension cannot access any other websites.

Who Uses This Extension

This extension is published as unlisted on the Chrome Web Store and is intended solely for use by employees of Traxis Manufacturing on company shop-floor computers. It is not marketed to or intended for public use.

Changes to This Policy

If we update this policy, the revised version will be posted at this URL. The extension's scope and data practices are unlikely to change, as it is an internal tool with a narrow, single purpose.
